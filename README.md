# Student Management System (Python + SQLite)

This is a beginner-friendly project using **Python + SQLite**.  
It supports full **CRUD operations** such as Add, List, Search, Update, and Delete.

## ✅ Features
- Add Student
- List Students
- Search by Student ID
- Update Student Details
- Delete Student
- Input validation (Year 1–5)
- UNIQUE Email validation
- Handles errors and exceptions

## ✅ Technologies Used
- Python 3
- SQLite Database (sqlite3 module)
- SQL Queries
- CLI (Command-line interface)

## ✅ How to Run the Project

1. Install Python 3
2. Download this repository
3. Open terminal (CMD)
4. Navigate to project folder:
5. Run the file:

python student_app.py

## ✅ Files Included
- `student_app.py` — Main Python application  
- `students.db` — SQLite database  
- `Student_Project_Report.pdf` — Project documentation
